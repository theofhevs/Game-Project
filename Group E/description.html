Our Project Game description

By Jonathan Araújo the 24/10/2023 at 10:00

In "The Tremendous Journey," players can choose from iconic characters like Bon Giorno, Tonio Fortebracci, Giuseppe Bucciarati, and Ari Vedercci, each with their own personality and unique style. While these heroes don't possess special powers, their determination and individual skills help them navigate the challenges of this captivating world.

The game features rich and varied environments inspired by Italy, blending medieval and fantastical aesthetics. Players traverse picturesque villages, climb majestic mountains, and explore lively alleyways. Each level provides immersion into Italian culture, with subtle references to cuisine, history, and art.

The ultimate goal of the game is to gather the ingredients needed to create the perfect pizza that will save the kingdom from imminent hunger. Unlike other games where customization is key, in "The Tremendous Journey," once all the ingredients are collected, the pizza is assembled, and the game is complete. Players must strategize to overcome obstacles, solve puzzles, and work together to ensure the success of this unforgettable culinary adventure.

The source code of the application is available on the github of the project.
