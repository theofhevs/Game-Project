Our Project Game description

By Jonathan Araújo the 24/10/2023 at 10:00

In "The Tremendous Journey," players can play Ari Vedercci . While this hero doesn't possess special powers, his determination and individual skills help him navigate the challenges of this captivating world.

The game features rich and varied environments inspired by Italy, blending medieval and fantastical aesthetics. Players traverse picturesque villages, climb majestic mountains, and explore lively alleyways. Each level provides various landscapes, with subtle references to cuisine, history, and art.

The ultimate goal of the game is to gather the ingredients needed to create the perfect pizza that will save the kingdom from imminent hunger. Unlike other games where customization is key, in "The Tremendous Journey," once all the ingredients are collected, the pizza is assembled, and the game is complete. Players must strategize to overcome 3 levels and a final boss in this unforgettable culinary adventure.

The source code of the application is available on the github of the project.
